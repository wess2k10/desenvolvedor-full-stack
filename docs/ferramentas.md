# HTML : Quais elementos foram usados?
nav : marcar um menu de navegação ; 
div : para dividir os containers ;
ul e li: para criar o menu de navegação da página com uma lista; 
h1 : título ;
h2 : subtítulo ;
img : para colocar uma imagem ;
p : criar um parágrafo ;
br : quebrar a linha ;
form : criar um formulário ;
label : criar um campo de texto ;
input : para criar campos de entrada de dados no formulário ;
select e option : escolher sua opcão do assunto(no contexto do meu projeto) ;
textarea : criar a área de texto ;
button : criar um botão ;
# CSS : Quais propriedades estão sendo usadas?
margin : definir a margem ;
padding : definir o espaçamento entre a borda de um elemento e o outro ;
box-sizing : para definir como o navegador calcula o tamanho total de um elemento ;
background-color : definir a cor de fundo ;
color : definir a cor de um elemento ;
font-size : definir o tamanho da fonte do texto ;
text-decoration : definir a decoração do texto ;
display :  definir como um elemento HTML é renderizado na página ;
gap : para criar um espaçamento uniforme entre itens de um layout ;
list-style : definir o estilo da lista ;
text-align : definir o alinhamento do texto ;
widht : definir a largura de um elemento ;
justify-content : é usado para alinhar itens flexíveis ao longo do eixo principal do contêiner ;
margin-botton : definir a margem do botão ;
align-items : defir o alinhamento do item ;
font-family : definir o estilo da fonte ;
font-weight : para definir o peso ou a espessura de um texto, sendo mais clara ou mais escura ;
align-self : para substituir o alinhamento padrão de um item flexível ou de grade, definindo sua posição no eixo transversal ;
position : definir a posição do elemento ;
top : é usada para posicionar um elemento a partir do seu ponto de referência, geralmente no topo ;
left : para definir a posição horizontal de um elemento, deslocando sua borda esquerda a partir de um ponto de referência ;

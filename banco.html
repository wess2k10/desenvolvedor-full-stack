Um banco de dados é uma coleção organizada de informações armazenadas
eletronicamente em um sistema de computador. Ele permite coletar, armazenar,
gerenciar e recuperar dados de forma eficiente e estruturada, usando um sistema de
gerenciamento de banco de dados (DBMS). Bancos de dados são usados para
diversas aplicações, como gerenciar dados de clientes, registros de hospitais ou
informações de usuários em redes sociais.
